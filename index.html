electerm is a terminal/ssh/sftp client(mac, win, linux) based on electron/node-pty/xterm and many other libs, it works like a combination of guake and xshell.

features

  • work as a terminal or ssh/sftp client(similar to xshell)
  • hotkey to bring window back to front(simliar to guake, default is ctrl + 2)
  • support multi platform(linux, mac, windows)
  • support multi-language(en, cn only for now)

user tips

  • double clicka ssh tab will duplicate a same tab, and enter same path as original tab(limited support)
  • double clickblank area of tab bar will open a new ssh terminal too
  • right clicka ssh tab you can open context menu, from context menu there areclose other tabs,close tabs on the right, and more function
  • by default,Ctrl + 2will bring electerm window to front, hotkey can be set insettingmenu
